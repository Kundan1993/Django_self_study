The firstof tag returns the first argument that is not an empty variable.
Empty variables can be an empty string "", or a zero number 0, or a boolean false.

{% firstof x y z %}

The for tag allows you to iterate over items in an object.
Objects can be array-like objects like a Python list or object-like objects like a Python dictionary:
    {% for x in fruits %}
  • {{ x }}
    • {% endfor %}
{% for x, y in mycar.items %}

The {{ x }} is {{ y }}.

{% endfor %}

forloop.counter: The current iteration, starting at 1.

    {% for x in fruits %}
  • {{ forloop.counter }}
    • {% endfor %}

forloop.counter0:The current iteration, starting at 0.

    {% for x in fruits %}
  • {{ forloop.counter0 }}
    • {% endfor %}

forloop.first: Check if this iteration is the first iteration.

    {% for x in fruits %}
  • {{ x }}
    • {% endfor %}

forloop.last:Check if this iteration is the last iteration.

    {% for x in fruits %}
  • {{ x }}
    • {% endfor %}

forloop.parentloop: Refers to the parent loop.

    {% for x in cars %}
  • {{ x }}:
      {% for y in colors %}
    • {{ forloop.parentloop.counter }} {{ y }}
      • {% endfor %}
    • {% endfor %}

forloop.revcounter:The current iteration, counting backwords, ending at 1.

    {% for x in fruits %}
  • {{ forloop.revcounter }}
    • {% endfor %}

forloop.revcounter0:The current iteration, counting backwords, ending at 0.

    {% for x in fruits %}
  • {{ forloop.revcounter0 }}
    • {% endfor %}

The ifchanged tag allows you to check a value in a loop and output a code if the value has changed since the last iteration. {% for x in cars1 %} {% ifchanged x.brand %}

{{ x.brand }}:

{% endifchanged %}

{{ x.model }}, {{ x.year }}

{% endfor %}

Loop through a list and write different messages if the value has changed or not:

{% for x in mylist %} {% ifchanged %}

New value: {{ x }}

{% else %}

Same value: {{ x }}

{% endifchanged %} {% endfor %}

The now tag inserts the current date and/or time, according to the specified format.

{% now "Y-m-d" %}

{% now "Y-m-d g:i:s" %}

Year, 2 digits (00-99)(y):-> {% now "y" %}

Year, 4 digits (0001-9999)(Y):-> {% now "Y" %}

Returns whether now is a leap year or not (True-False)(L):-> {% now "L" %}

Month, 2 digits (01-12)(m):-> {% now "m" %}

Month, 1 or 2 digits (1-12)(n):-> {% now "n" %}

Month, 3 letters (Jan-Dec)(M):-> {% now "M" %}

Month, 3 lower case letters (jan-dec)(b):-> {% now "b" %}

Month, text in local language(E): {% now "E" %}

Month, full text (January-December)(F):-> {%now "F" %}

Month, max 5 letters, if month name has more, use proper abbreviation (Jan. Dec.)(N): {% now "N" %}

Number of days in month (28-31)(t):- {% now "t" %}

Day of month (01-31)(d): {% now "d" %}

Day of week (Sun-Sat)(D): {% now "D" %}

Day of month (1-31)(j): {% now "j" %}

Day of week (Sunday-Saturday)(l): {% now "l" %}

Ending of number (st-nd-rd-th)(S): {% now "S" %}

Day of week (0-6)(w): {% now "w" %}

Day of year (1-366)(z): {% now "z" %}

Week of year (ISO-8601) (1-53)(W): {% now "W" %}

Name of time zone, like (GMT) or (UTC)(e): {% now "e" %}

The resetcycle tag is used inside a cycle, and resets the cycle, making it start at the beginning. It does not reset the loop, only the cycle.
    {% for x in fruits %}
  • {{ x }}
    • {% if x == "Banana" %} {% resetcycle %} {% endif %} {% endfor %}
    {% for x in fruits %}
  • {{ x }}
    • {% if x == "Banana" %} {% resetcycle mybg %} {% endif %} {% endfor %}

The spaceless tag is used to remove any space between tags, in the code.
The spaceless tag removes any whitespaces, new lines and tabs. {% spaceless %}
    {% for x in fruits %}
  • {{ x }}
    • {% endfor %}
{% endspaceless %}
