floatformat
Round a number to only two decimals: {{ mynumber|floatformat:2 }}

{{ 7.122489|floatformat:2 }}

{{ 7.1|floatformat:2 }}

{{ 7|floatformat:2 }}

If the argument value is a negative number, the value will be displayed with the specified number of decimals, only if the value has decimals:

{{ 7.122489|floatformat:-2 }}

{{ 7.1|floatformat:-2 }}

{{ 7|floatformat:-2 }}

You can add a g in the argument value to specify that the result should include a separator for each thousand:

{{ mynumber1|floatformat:"2g" }}

The time filter formats and return the time from a datetime object.

You can specify the format by using these format characters:

g---> Hour (1-12)
G---> Hour (0-23)
h---> Hour (01-12)
H---> Hour (00-23)
i---> Minutes (00-59)
s---> Seconds (00-59)
u---> Microseconds (000000-999999)
a---> am/pm suffix (a.m. or p.m.)
A---> AM/PM suffix (A.M. or P.M.)
f---> Time, hour and minute, but minutes are left out if they are 0
P---> Same as f, but also including a.m. or p.m. and:
If time is exactly "12:00 a.m." it returns "midnight".
If time is exactly "12:00 p.m." it returns "noon".

Current Time:-> {{ mydate|time }}

Current Time Hours(1-12)(g):-> {{ mydate|time:'g' }}

Current Time Hours(1-23)(G):-> {{ mydate|time:'G' }}

Current Time Hours(01-12)(h):-> {{ mydate|time:'h' }}

Current Time Hours(00-23)(H):-> {{ mydate|time:'H' }}

Current Time Minutes (00-59)(i):-> {{ mydate|time:'i' }}

Current Time Seconds (00-59)(s):-> {{ mydate|time:'s' }}

Current Time am/pm(a):-> {{ mydate|time:'a' }}

Current Time AM/PM(A):-> {{ mydate|time:'A' }}

Time, hour and minute, but minutes are left out if they are 0(f):-> {{ mydate|time:'f' }}

ame as f, but also including a.m. or p.m. and(P):-> {{ mydate|time:'P' }}
